Annotation
1@Test -> Used to declare any method as a test case
2@Before -> Used to execute any method before any test case
3@After -> used to execute any line of code after running a used test case 
4@Ignore -> used to ignore any line of code during the test case 
5@BeforeClass -> to execute a statement before testing the cases
6@AfterClass -> to execute a stetement after testing a case
